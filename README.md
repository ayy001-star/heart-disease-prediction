# Heart Disease Prediction

## Overview
A machine learning project that predicts heart disease using the UCI Heart Disease dataset.

# Models Used
- Random Forest (Best - AUC: 0.909)
- Linear Regression,

# Features
- Feature correlation analysis
- ROC curve comparison
- SHAP explainability (waterfall + summary plots)

## How to Run
1. Clone the repo
2. Install dependencies: pip install -r requirements.txt
3. Open heart_disease.ipynb in Jupyter

## Dataset
UCI Heart Disease Dataset via Github
